# Menu PDF

This directory contains the PDF file for your restaurant menu.

## File Required

- `menu.pdf` - Your complete restaurant menu

## How to Add Your Menu

1. **Create a PDF**: Use any PDF creation tool to create your menu
2. **Place in this Directory**: Save the PDF file as `menu.pdf` in this folder
3. **That's it!** The menu will automatically appear on your website

## Recommended Tools

- **Canva** (recommended): https://www.canva.com/create/menus/
  - Free menu templates
  - Professional designs
  - Quick and easy
  
- **Microsoft Word**: Create document and export as PDF
- **Google Docs**: Create document and download as PDF
- **LibreOffice**: Free alternative to Word
- **Adobe InDesign**: Professional design tool

## Steps to Create Menu

1. Choose one of the tools above
2. Create your restaurant menu with items and prices
3. Export/Download as PDF
4. Rename to `menu.pdf`
5. Save to this folder (`public/menus/`)

## File Size Tips

- Keep PDF under 5MB for best performance
- Use Canva or online PDF compressors if file is too large
- Compress images before converting to PDF

Visit http://localhost:8000/menu to see your menu!
